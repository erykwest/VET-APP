# 05 — User Flows e Core Loop

## Core loop principale
Profilo pet → domanda → risposta utile → memoria → reminder → ritorno

## Flow 1 — Attivazione
### Obiettivo
Portare l'utente dal primo accesso alla prima percezione di valore nel minor tempo possibile.

### Passi
1. Registrazione / login
2. Benvenuto breve
3. Inserimento profilo minimo del pet
4. Invito alla prima domanda
5. Prima risposta
6. CTA: salva, continua o imposta reminder

### Rischi
- onboarding troppo lungo
- richiesta di troppi dati prima del valore
- prima risposta percepita come generica

## Flow 2 — Uso ricorrente
### Obiettivo
Far tornare l'utente per bisogni diversi mantenendo continuità.

### Passi
1. L'utente rientra
2. Vede storico / stato pet / reminder
3. Riprende un tema o ne apre uno nuovo
4. Riceve risposta contestualizzata
5. Aggiorna note o reminder

### Rischi
- dashboard vuota o poco utile
- impossibilità di ritrovare il contesto
- reminder scollegati dalla percezione di utilità

## Flow 3 — Reminder-driven reactivation
### Obiettivo
Usare il reminder come driver di ritorno e non come funzione isolata.

### Passi
1. Reminder visibile / notificato
2. Apertura app
3. Contesto del pet richiamato
4. Possibilità di fare domanda collegata
5. Aggiornamento stato / nuova scadenza

## Flow 4 — Caso sensibile
### Obiettivo
Gestire bene i casi in cui il prodotto deve mantenere fiducia e prudenza.

### Passi
1. Domanda percepita come sensibile
2. Risposta chiara ma non prescrittiva
3. Indicazione di attenzione / approfondimento / supporto professionale
4. Salvataggio della conversazione

## Esperienza da evitare
- chatbot anonimo che non ricorda nulla
- risposta standard che sembra copiata
- app che promette tutto ma non guida l'utente
- reminder senza contesto

## Checklist PM del core loop
- Il valore emerge entro la prima sessione?
- L'utente capisce perché creare il profilo pet?
- Lo storico è davvero utile o solo presente?
- Il reminder aumenta il ritorno o è decorativo?
- L'esperienza può essere spiegata in una singola frase semplice?
